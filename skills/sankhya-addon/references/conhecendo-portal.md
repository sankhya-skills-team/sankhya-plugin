# Portal do Desenvolvedor Sankhya — Visão geral

> Fonte oficial: https://developer.sankhya.com.br/docs/conhecendo-o-portal

Base de conhecimento técnico oficial das APIs e frameworks proprietários da
Sankhya: artigos, tutoriais, exemplos de código e recursos para download.

## Frameworks de desenvolvimento

- **Add-on Studio** — desenvolvimento moderno de extensões (recomendado p/ addon novo).
- **SDK Sankhya** — serviços de backend (controller, injeção, transação; ver `sdk-*.md`).
- **SankhyaJS** — frontend HTML5 (ver `sankhya-js*.md`).
- **Dicionário de Dados / ORM (JAPE)** — ver `jape.md`, `acesso-dados.md`, `dicionario-de-dados.md`.

## Tipos de personalização

- **Add-ons** — abordagem moderna recomendada.
- **Módulos Java** — regras na Central (skill `sankhya-modulo-java`).
- **Migração de DWF legado** — ver `migracao-dwf-addon.md`.

## Organização do conteúdo

- **Guias** — documentação fundamental.
- **Recipes** — exemplos práticos de implementação.
- **API Reference** — especificações técnicas.
- **Tech Academy** — trilhas de certificação e treinamento.

## Recursos do ecossistema

Contribuição da comunidade, certificação profissional, integração com Marketplace
para publicar soluções, debug e logging remotos, documentação de boas práticas.
